Insta-informer

Proof of concept to understand how to interact with Instagram due to the update to the new API v2.0, which only allows you to interact if you are a business or professional account.

The basic v2.0 API only allows you to perform certain methods.

Because of this we interact directly as if we were a browser, without API and with cookies and CSRF, necessary to disable second factor authentication.

So with this tool you can search who doesn't follow you back on Instagram.
